ソースを作成し右上の`再生ボタン`で実行。
```:hello2.c (SJISにて作成)
#include <stdio.h>
int main(void)
{
    printf("こんにちは！C言語\n");
    return 0;
}
```

```output:出力結果
PS C:\workspace\gcc> cd "c:\workspace\gcc\" ; if ($?) { gcc hello2.c -o hello2 } ; if ($?) { .\hello2 }
こんにちは！C言語
```
