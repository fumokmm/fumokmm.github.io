### 1. コミット後、ひとまず作業用のブランチを作ります
作業用ブランチ名は`work`とします。
```
$ git switch -c {em{work}em}
```

### 2. 次に戻りたい地点まで、resetコマンドで戻ります
戻りたい地点を`xyz`とします。
```
$ git reset {em:blue{--mixed}em} {em{xyz}em}
```
- ここでのポイントは`--mixed`を指定することです。
- `reset`のデフォルトは`--mixed`なので、特に指定しなくても良いですが、ここでは敢えて意識するため指定しています。
```:単純にこれでもOK
$ git reset {em{xyz}em}
```
