```output
$ gradle writeProp {em{-PpropA=100}em}
```

複数指定する場合は`-P`や`--project-prop`を連ねて書きます。
```output
$ gradle writeProp {em{-PpropA=100}em} {em{-PpropB=200}em} {em{--project-prop propC=300}em}
```
