```syntax:SQL
coalesce(文字列, [, 文字列...])
```

```:使用例
select coalesce({em{'ABC'}em}, {em:blue{null}em});  {em:comment{-- 'ABC'}em}
select coalesce({em:blue{null}em}, {em{'ABC'}em});  {em:comment{-- 'ABC'}em}
select coalesce({em:blue{null}em}, {em:blue{null}em}, {em{'ABC'}em}, 'DEF');  {em:comment{-- 'ABC'}em}
select coalesce({em:blue{null}em}, {em:blue{null}em}, {em:blue{null}em});  {em:comment{-- null}em}
```
- さすがに全部`null`だと`null`が返却されます。
