`string_agg`の中で`order by`を使ってデータの並び順を指定します。

```:SQL
select
    string_agg(dat, '/' {em{order by dat desc}em})
from (
    select 'a' as dat
    union
    select 'b' as dat
    union
    select 'c' as dat
) as dat_table
```

```:取得結果
c/b/a
```
