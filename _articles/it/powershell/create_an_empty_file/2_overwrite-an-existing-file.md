`-Force`オプション付けると、既存ファイルがあった場合上書きします。
<div class="code-box-syntax">
<div class="title">空ファイルを作成する(既存ファイルがあった場合上書き)</div>
<pre>
New-Item &lt;ファイルのパス&gt; <em>-Force</em>
</pre>
</div>
