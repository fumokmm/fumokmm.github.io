### 可変長引数

可変長引数は変数`$args`という配列に渡されます。
```:variable_length_parameter.ps1
function foo {
    Write-Host {em{$args}em}.Count  {em:comment{# 引数の数を出力}em}
    foreach($i in {em{$args}em}) {
        Write-Host $i
    }
}

# 呼び出し
foo a b あいう
```

```output:出力結果
PS C:\temp> .\variable_length_parameter.ps1
3
a     
b     
あいう
```
