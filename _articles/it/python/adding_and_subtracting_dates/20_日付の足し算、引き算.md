```
import datetime

now = datetime.datetime.now()
print(type(now))  {em:comment{# => <class 'datetime.datetime'>}em}
print(now)        {em:comment{# => 2022-02-08 21:46:45.976739}em}

{em:comment{# 日付1日の増分}em}
{em{delta}em} = datetime.timedelta(days=1)
print(type(delta))  {em:comment{# => <class 'datetime.timedelta'>}em}
print(delta)        {em:comment{# => 1 day, 0:00:00}em}

tomorrow = now {em{+ delta}em}
print(type(tomorrow))  {em:comment{# => <class 'datetime.datetime'>}em}
print(tomorrow)        {em:comment{# => 2022-02-09 21:46:45.976739}em}

yesterday = now {em{- delta}em}
print(type(yesterday))  {em:comment{# => <class 'datetime.datetime'>}em}
print(yesterday)        {em:comment{# => 2022-02-07 21:46:45.976739}em}
```

```output:出力結果
<class 'datetime.datetime'>
2022-02-08 21:46:45.976739
<class 'datetime.timedelta'>
1 day, 0:00:00
<class 'datetime.datetime'>
2022-02-{em{09}em} 21:46:45.976739
<class 'datetime.datetime'>
2022-02-{em{07}em} 21:46:45.976739
```
- 1つ目の日付がnow(今日)
- 2つ目の日付がtomorrow(明日)
- 3つ目の日付がyesterday(昨日)
