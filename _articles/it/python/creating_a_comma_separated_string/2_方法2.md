ただし、[方法1](#方法1)の方法だと文字列が2文字以上の繰り返しのときにうまく行かないので、以下のようにリストに`*`するほうが`list()`も省略できてスマートです。
```
params = ", ".join(["?"] * 10)
print(params)
```
```output
?, ?, ?, ?, ?, ?, ?, ?, ?, ?
```

### リスト * 回数
```syntax
リスト * n
```
- 指定のリスト要素をn回繰り返した新しいリストを返します。
- 上記例だと `['?', '?', '?', '?', '?', '?', '?', '?', '?', '?']` となります。
- リストに対する`*演算子`については、[こちら]({% link _it_articles/python/list.md %}#*演算子)を参照。
