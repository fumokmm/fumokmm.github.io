### 通常のクラス定義
```syntax
class クラス名(継承元クラス名):
    """クラスのドキュメンテーション"""
    def method1(self, ...):
       処理
       　：
       　：

    def method2(self, ...):
       処理
       　：
       　：
```

### 空クラス
```syntax
class クラス名:
    pass
```
