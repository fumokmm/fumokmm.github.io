`print`を使います。

```:hello1.py
print("Hello, World")
print("Hello, ")
print("World")
```

```output:実行結果
{em:command{$ python hello1.py}em}
Hello, World
Hello, 
World
```

普通に呼び出すと、改行コード付きで出力されます。 改行コードなしにするには、`end=""` を付けます。

```:hello2.py
print("Hello, ", {em{end=""}em}) {em:comment{# end="" とすると改行しない}em}
print("World")
```

```output:実行結果
{em:command{$ python hello2.py}em}
Hello, World
```
