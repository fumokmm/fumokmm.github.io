- python3.8以降で利用できます。
- 代入するときに`=`ではなく`:=`を使って代入します。
  - 形がセイウチの目と牙(:=)に似ていることから、「セイウチ演算子」と呼ばれているそうです。
- 代入した結果を値として評価したい場合に利用します。
```:例
i = 5
while (i {em{:=}em} i - 1):
    print(i)
```
```:出力結果
4
3
2
1
```
