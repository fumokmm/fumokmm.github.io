リストに要素が含まれているかをチェックするには、`in演算子`を利用します。

```syntax
<要素> in <リスト>
```
- `<要素>`が`<リスト>`に含まれている場合、`True`が、含まれない場合`False`が返却されます。

### 例
```
list = ["A", "B", {em{"C"}em}, "D", "E"]
assert ("C" {em{in}em} list) == {em:blue{True}em}
```
