インストール済みのパッケージを確認するには`list`か`freeze`コマンドを使います。  
両者は表示のされ方がちょっと違います。

```output:list
$ pip {em{list}em}
Package                Version
---------------------- --------------------
astroid                2.4.2
attrs                  19.3.0
Automat                0.8.0
beautifulsoup4         4.9.3
blinker                1.4
   ：
   ： (省略)
   ：
```

```output:freeze
$ pip {em{freeze}em}
astroid==2.4.2
attrs==19.3.0
Automat==0.8.0
beautifulsoup4==4.9.3
blinker==1.4
   ：
   ： (省略)
   ：
```
