以下のような構文で宣言します。
```syntax
{em{DECLARE}em} {em:blue{@変数名}em}m {em{TABLE}em} (
    <テーブル定義>
        :
        :
) 
```
以降、`@変数名`で定義した変数名をテーブルとして利用できるようになります。

### 例
```
{em{DECLARE}em} {em:blue{@id_tables}em} {em{TABLE}em} (id nvarchar(16))
```
