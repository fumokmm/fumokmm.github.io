まずはドキュメント内の文字列を一括置換するコードです。
```
Option Explicit

Sub ReplaceTexts()
    With ActiveWindow.Selection.Find
        {em{.Text = "あ"}em}
        .MatchFuzzy = False  {em:comment{' ワイルドカードを利用するときはあいまい検索をオフにする}em}
        .MatchWildcards = True  {em:comment{' ワイルドカード指定で検索}em}
        {em:comment{' 検索を実行}em}
        Do While .Execute
            Dim p_foundStr As String
            p_foundStr = ActiveWindow.Selection.Range.Text
            Debug.Print p_foundStr
            {em:comment{' 置換を実行}em}
            {em:blue{ActiveWindow.Selection.Range.Text = "●"}em}
        Loop
    End With
End Sub
```
- 上記は`"あ"`という文字列を`"●"`に置換しています。
