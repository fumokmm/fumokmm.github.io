### #set
`#set`は右辺の値を左辺に代入する指示子です。

```
#set( $value = "$value * 3")
#set( $name = "Yojigen")
#set( $name = 'Yojigen')
```
- ダブルクォーテーションを使っている場合は評価され、シングルクォーテーションを使っている場合は評価されません。

```
#set($name = "ふも")
#set($a = "$name")
#set($b = '$name')
$a
$b
```

```output:処理結果
ふも
$name
```
